# Метрики классификации

[Назад к списку компонентов](../README.md)

## Назначение

Позволяет рассчитать ряд метрик для оценки качества классификации.

## Алгоритмы

Точность = TP / (TP + FP)

Полнота = TP / (TP + FN)

F1-мера = 2 * (Точность * Полнота) / (Точность + Полнота)

Accuracy = (TN + TP) / (TN + FP + FN + TP)
